## About TransMov

TransMov is a macOS application for video and animation processing, providing:

- **Animation Preview**: Preview and play APNG, GIF, and WebP animation files
- **Video Remove BG**: Convert videos to animations with AI-powered background removal
- **Video > Animation**: Direct video to animation conversion
- **Animation ↔ Animation**: Convert between APNG, GIF, and WebP formats
